----- Variable Data Type -----

Variable can containt, numbers and texts. 
• Text must go with double or single quotes "", the data type it is called string.
• Number doesn´t contain quote.

Declare and assign a variable
1. To declare a variable, you have to put "var".
Example: var name_variable

2. To assign a variable, use the equal sign =.
Example: var name_variable = 2;

3. You can declare and assign a variable:
Example: var car = "ford";

Once statement many variable
Example: var person ="Juan", age = 32, state = "Idaho";

Variable undefined
Example: var age;

Redeclaring variable
Example: var state = "Idaho";
